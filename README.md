# Cine Aurora | Vintage Cinema Experience

Proyecto de desarrollo web creado para el Trabajo Práctico N°2. Consiste en la maquetación avanzada de una plataforma cinematográfica con estética retro, aplicando técnicas de diseño moderno como Flexbox y CSS Grid, con un enfoque estrictamente modular.

## Descripción del Proyecto

El sitio web está diseñado para "Cine Aurora", un cine con una propuesta visual inspirada en la cartelera clásica de los años 50 y 60. La plataforma permite a los usuarios explorar una cartelera de clásicos del cine, consultar próximos lanzamientos, ver detalles técnicos de cada película y registrarse mediante formularios validados. El sistema destaca por su consistencia visual y su capacidad de adaptarse a cualquier dispositivo.

## Tecnologías y Conceptos Aplicados

* **HTML5 Semántico:** Estructura organizada para mejorar la accesibilidad y el SEO del catálogo de cine.
* **CSS3 Avanzado:**
    * **Modularización:** Uso intensivo de `@import` para separar estilos por componentes (`nav.css`, `card-vertical.css`, `button.css`, etc.).
    * **Variables CSS (`:root`):** Centralización de la paleta de colores y tipografías.
    * **Flexbox & Grid:** Implementación de layouts para las tarjetas de películas y el sistema de navegación fluido.
    * **Media Queries:** Diseño responsivo para garantizar la correcta visualización en móviles, tablets y escritorio.
* **Google Fonts:** Integración de las fuentes *Bebas Neue*, *Montserrat* y *Pacifico* para lograr el acabado de pincel y póster clásico.
* **Git & GitHub:** Trabajo colaborativo mediante un flujo de ramas (`alumno-nombre`), Pull Requests y resolución de conflictos para lograr un código unificado.
